Codeunit "WandP TRC Del Note Management"

ID 70504

Codeunit handles the creation and sending of delivery note emails for sales orders.

Remarks

This codeunit provides functionality to:

  • Format and send emails for delivery notes
  • Configure email subject lines using customizable templates
  • Handle email sending in background to avoid transaction locks

Developer Note: This codeunit contains the core email functionality that was extracted from the subscriber codeunit to improve separation of concerns. This makes the code:

  • More maintainable (email logic is centralized)
  • More testable (can be called directly without events)
  • More reusable (can be called from other places if needed)
