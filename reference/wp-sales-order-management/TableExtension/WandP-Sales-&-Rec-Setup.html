TableExtension "WandP Sales & Rec Setup"

ID 70500
Extends: Microsoft.Sales.Setup."Sales & Receivables Setup"

Extends Sales and Receivables Setup to add delivery note email configuration.

Remarks

This extension adds two fields:

  • TRC Delivery Note Email To: The recipient email address for delivery notes
  • TRC Delivery Note Subject: Customizable subject line with %1 placeholder for order number

The email configuration is used by the TRC Delivery Note Management codeunit when sending delivery note emails for sales orders with delivery comments.

Fields

Name Type Description
"WandP TRC Del Note Email To" Text[250]

Specifies the email address where delivery notes should be sent.
"WandP TRC Del Note Subject" Text[250]

Specifies the subject line template for delivery note emails.
"WandP Ship Complete post Ship" Boolean

If enabled, the sales order will be marked as shipped complete after posting a shipment.
